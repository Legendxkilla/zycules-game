# Zycules Ascends Olympus

## Setup
1. Clone this repo:
   ```bash
   git clone https://your.git.repo/zycules-game.git
   cd zycules-game
   ```
2. Add your asset files in `assets/`.
3. Deploy to Vercel (or any static host):
   ```bash
   vercel --prod
   ```

## Play
Visit the deployed URL to run the game in browser.
